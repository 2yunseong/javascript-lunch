# javascript-lunch

우아한테크코스 레벨1 점심 뭐 먹지 미션

배포 주소 : https://bassyu.github.io/javascript-lunch/

## 시작하기

### 기본 설정

```
git clone https://github.com/bassyu/javascript-lunch.git
cd javascript-lunch
```

### step1

#### 설정

```
git checkout -b step1
git pull origin step1
npm ci
```

#### 테스트

```
npm run test-unit
```

#### 실행

```
npm run start
```

#### 빌드

```
npm run build
```
